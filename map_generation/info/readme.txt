"cities.csv" contains the city number, and the tile numbers that are connected to
the city

"dice_value_info.csv" contains the number of times each dice value occurs in
total between all tiles in the game

"landscape.csv" contains which tiles are water tiles, and which are to be
randomly set(corresponding to the setup in catan.png)

"resource_info.csv" contains the number of each resource that are randomly
spread between the randomly generated tiles
